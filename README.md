# Appium Mobile Automation Project 📱🚀

This project is a comprehensive guide and implementation of a **Mobile Automation Script** using Appium for an Android Calculator application.

## 🛠️ Tech Stack & Tools
* **Language:** Java
* **Automation Tool:** Appium Server 2.0
* **Driver:** UiAutomator2
* **Build Tool:** Maven
* **IDE:** IntelliJ IDEA
* **Inspector:** Appium Inspector

## 📌 Features
* Automated basic calculator operations.
* Use of **Desired Capabilities** for device connection.
* Efficient element locating using **Resource IDs**.
* Handling environment setup and common errors like `ANDROID_HOME`.

## 🚀 How to Run
1. Install **Node.js** and **Appium**.
2. Install the **UiAutomator2** driver.
3. Open the project in **IntelliJ**.
4. Start the **Appium Server**.
5. Run the `CalculatorTest.java` file.
